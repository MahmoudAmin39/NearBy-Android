# NearBy-Android

A sample Android application for showing near by places to user using Foursquare API

# Technologies used: 
- Kotlin
- ViewModel
- LiveData
- Room
- Retrofit
- Glide
# App main Architecture:
- Model View ViewModel (MVVM)
